# Astra Vision

Astra is designed for AI-native artifact-driven development.

The main idea is that software systems should be described through first-class artifacts instead of class hierarchies and implicit runtime behavior.

Key first-class artifacts:

- schema
- command
- event
- query
- workflow
- projection
- policy
- capability
- api
- view
- invariant

The goal is to make software understandable to humans, compilers, and AI systems at the same time.
